Available APIs

  • GET /api/health - Health check
  • GET /api/handlers - List registered handlers
  • GET /api/schedules - List active schedules
  • POST /api/schedule - Create new schedule
  • POST /api/run_now - Execute handler immediately
  • GET /api/executions - View execution history
  • Full API Documentation →

Setting Up the Portal

  1. Create portal_config.yaml in project root
  2. Define portal root, microsites, and component libraries
  3. Build frontend with Vite in schedule-zero-islands
  4. Restart server to load full portal
